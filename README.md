# Uso-del-Metodo-de-Busqueda-Binaria
Programa en Python que lee un archivo con 50,000 números, los ordena y permite buscar un valor usando búsqueda binaria. Solicita un número al usuario, lo localiza en la lista y mide el tiempo de búsqueda, mostrando si fue encontrado y su posición.

##  Requisitos
- Python 3.x
- Archivo `datos.txt` con números

---

##  Archivos
- `programa.py` → código principal  
- `datos.txt` → contiene los números  
- `README.md` → documentación  

---

##  Cómo ejecutar
1. Coloca `datos.txt` en la misma carpeta
2. Ejecuta el programa:
3. Ingresa el número a buscar

---

##  Algoritmo utilizado
Búsqueda binaria:
- Divide la lista en mitades
- Reduce el número de comparaciones
- Requiere datos ordenados

---

##  Complejidad
- Ordenamiento: O(n log n)  
- Búsqueda binaria: O(log n)

---

##  Resultado
El programa muestra:
- Si el número fue encontrado
- Posición en la lista
- Tiempo de búsqueda
